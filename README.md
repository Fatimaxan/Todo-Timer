# قائمة مهام مع مؤقت

البرنامج عبارة عن **قائمة مهام (To-Do List) مزودة بمؤقت**!  
يعمل المؤقت بالألوان، بدءًا من **الأحمر** مرورًا بـ **البرتقالي** و**الأصفر** وصولًا إلى **الأخضر**.  
يمكنك **عكس ترتيب الألوان** بالضغط على زر الألوان الموجود في أعلى يمين الشاشة.

---

## الأزرار ووظائفها

- **▶️ تشغيل** : يبدأ المؤقت لكل مهمة.  
- **✅ تم** : تم إتمام المهمة (يمكنك الضغط عليها حتى قبل انتهاء الوقت).  
- **⏭️ تخطي** : تخطي المهمة؛ ستصبح في نهاية الطابور لتُنجز لاحقًا.  
- **➕ إضافة** : إضافة مهام جديدة ورؤية الطابور.  
- **👁️ الرؤية** : إخفاء أو إظهار المؤقت.
